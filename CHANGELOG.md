# Historial de cambios

## v1.0
- Generación inicial de catálogos digitales.
- Carga de productos desde Excel.
- Relación de productos con imágenes.
- Exportación de catálogo para uso comercial.

## v1.1
- Publicación de catálogos en hosting.
- Configuración de rutas públicas.
- Preparación de archivos web para vendedores y clientes.

## v1.2
- Panel administrativo.
- Gestión de usuarios.
- Gestión de vendedores.
- Flujo de pedidos comerciales.
- Integración con correo para envío de pedidos.

## v1.3
- Mejoras en experiencia de vendedores.
- Ajustes para publicación en hosting.
- Validaciones de enlaces y catálogos publicados.

## v1.4
- Propuesta de modo híbrido inteligente para imágenes.
- Posibilidad de usar imágenes locales o imágenes alojadas en servidor externo.
- Preparación del sistema para futuras mejoras sin romper el flujo actual.

## Notas generales
Este historial documenta la evolución del sistema y sirve como respaldo técnico del proceso de desarrollo.
