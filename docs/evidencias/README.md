# Evidencias de desarrollo

Esta carpeta puede usarse para guardar respaldos externos del proceso de desarrollo.

Evidencias recomendadas:

- Capturas de pantalla del sistema.
- Videos probando el sistema.
- Versiones ZIP del proyecto.
- Documentos de propuesta.
- Capturas de commits.
- Capturas de conversaciones donde se explica la idea y evolución.
- Manuales de uso.
- Pruebas de funcionamiento.

Nota:
No subir contraseñas, claves SMTP, API keys reales ni información sensible.
